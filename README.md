# Sanskriti
Team Nuvs - Sanskriti
===================
 To run the application
 ---------------------
* Make sure Metamask is installed and configured to Mumbai Testnet RPC
* On the terminal\
  -> cd dapp-ui\
  -> npm i\
  -> npm run dev
* On a new terminal\
  -> cd others\
  -> npm i\
  -> npm start
