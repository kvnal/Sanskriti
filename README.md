# Sanskriti
Team Nuvs - Sanskriti
===================
 To run the application
 ---------------------
* Make sure Metamask is installed and configured to Mumbai Testnet RPC
* On the terminal\
  -> cd dapp-ui\
  -> npm i\
  -> npm run dev
* On a new terminal\
  -> cd others\
  -> npm i\
  -> npm start

Working
------
* The application Dashbord works on Web3 js with Nuxt Framework and Vue.
* Through Web3 we can connect and call funtions of our Sanskriti.sol contract.
* The contract is deployed on the Polygon Mumbai RPC testnet, also Metamask is configured with Polygon Mumbai RPC Testnet.
* Product enlisting, buying, selling all occur on the Blockchain through our contract.   
* Since our aim was to help the underpriveledged through this application we have also included INR and Card mode of payment for ease of use
* Also binance API is used to get realtime exchange and easy buying of Etherium coin.
* Face ID runs on central server to reduce total network cost. Effective AI/ML integration helps us in mathing the face capturesd by webcam to the UID image, also tesseract helps in returning the UID number which can be validated by administraion for security purposes.
* We have made the D-App extremely user firendly and reduced most of the hassles of transacting on blockchain.
* We have also included Contact us so that any user can reachout to the support team for assistance.
